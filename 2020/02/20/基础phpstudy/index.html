织梦cms的搭建

安装phpstduy

先下载链接> https://www.xp.cn/

直接下载windows版本就行，解压安装包安装即可

配置cms

在首页开启Apache功能

在网站里创建新网站，域名可以自定义，第二域名可以不填，根目录默认是./phpstudy_pro/www/[网站域名]

这里可以自定义根目录

织梦cms官网下载，选择UTF-8版本，解压

复制uplode文件夹下的文件，粘贴到网站的根目录下

打开网站，根据自己需要配置即可

mysql安装

phpstudy的环境安装可以直接下载

打开控制台,进入mysql的根目录

mysql -u root -p然后输入数据库密码进入mysql
